# Excel-Amazon-Sales-Dashboard
Interactive Amazon Sales Dashboard built in Microsoft Excel using Pivot Tables, Pivot Charts, Slicers, and Data Cleaning techniques.

Amazon Sales Performance Dashboard

Project Overview

This project analyzes Amazon sales data using Microsoft Excel to uncover sales trends, product performance, regional insights, and delivery metrics. An interactive dashboard was created using Pivot Tables, Pivot Charts, and Slicers to support business decision-making.

---

Objectives

- Clean and prepare raw sales data
- Analyze sales performance across regions
- Compare product category sales
- Monitor delivery performance
- Build an interactive Excel dashboard

---

Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Conditional Formatting

---

Data Preparation

- Removed duplicate records
- Corrected inconsistent data
- Formatted dates and currencies
- Cleaned missing values
- Organized data for Pivot Tables

---

Dashboard Features

- Sales Amount by Product Category
- Cancellation Rate by Region
- Delivery Status Analysis
- Total Sales by Region
- Effective Sales
- Average Delivery Days
- Total Revenue KPI
- Fast Delivery Percentage
- Highest Selling Product Category
- Top Performing Regions
